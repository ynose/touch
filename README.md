# touch
Windows用ファイルタイムスタンプ更新ツール

更新日時のみ更新します。  
作成日時やアクセス日時は更新されません。

## 使い方
#### コマンドラインから使う場合
touch.batとtouch.wsfを置いたディレクトリをPATH環境変数に追加してください。
```
TOUCH sample.txt "2015/07/03 15:00:00"
```
日付を省略するとPC日付に更新されます。  

---
#### ドラッグ＆ドロップで使う場合
**touch.wsf** のアイコン(ショートカットでも可)にファイルをドラッグ＆ドロップすると、PC日付に更新されます。
